
# cli-weather

## Introduction
1 An application used to know the current weather in a city.

## Libraries used
1. 'meow'
1. 'chalk'
1. 'update-notifier'
1. 'yql'
1. 'lodash'

## Install
1. $ npm install meow
1. $ npm install chalk
1. $ npm install update-notifier
1. $ npm install yql
1. $ npm install lodash

## Usage
```sh
$node cli.js weather <city> <country> <scale>
```


## Example:
```sh
$node cli.js weather London
```
```sh
weather, London
Condition: Partly Cloudy
Temperature: 9C
```
